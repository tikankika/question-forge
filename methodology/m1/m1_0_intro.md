# M1: CONTENT ANALYSIS
## Dynamic Dialogue Process for Understanding Conducted Instruction

**Classification:** 🔷 AI-Facilitated with Teacher Co-Creation
**Duration:** 2.5-3.5 hours
**Process Nature:** Structured dialogue with adaptive inquiry
**Primary Purpose:** Transform conducted instruction into validated learning objectives
**Primary Output:** Learning objectives with evidence + supporting documentation

---

## FUNDAMENTAL PRINCIPLES

### What M1 Does
**M1 (Content Analysis) DESCRIBES what was taught.**
**M2 (Assessment Planning) DESIGNS how to assess it.**

This module focuses exclusively on understanding conducted instruction through systematic analysis and dynamic dialogue. All assessment design decisions (Bloom's distributions, question counts, difficulty calibration) belong in M2.

### Dialogue, Not Interrogation
M1 is a **conversation** between Claude and teacher, not a rigid checklist. Claude adapts questions based on responses, probes interesting details, and follows the natural flow of dialogue while maintaining systematic coverage.

### Role Separation
This document contains instructions for **both Claude and teachers**, clearly marked:
- **FOR CLAUDE:** How to facilitate the dialogue
- **FOR TEACHERS:** What to expect and how to participate

## CONCEPTUAL FOUNDATION

### The Challenge
Teachers possess deep implicit knowledge about their instruction—what they emphasised, why certain examples worked, where students struggled. But this knowledge is often tacit, not explicitly documented. M1 externalises this knowledge through structured dialogue.

### The Process
Through six stages (Stage 0–5), Claude and teacher collaborate to create:
1. **Content architecture** - What topics, in what priority order
2. **Example catalogue** - Which illustrations worked best
3. **Misconception registry** - Common student errors
4. **Scope boundaries** - What's in vs. out for assessment
5. **Learning objectives** - Specific, measurable, evidence-based

### Process Flow
The stages are designed to build systematically: Stage 0 provides initial analysis for Stage 1 validation, which establishes the foundation for Stage 2's deeper inquiry. Stages 3 and 4 follow in sequence — documenting examples, then misconceptions — and inform Stage 5's synthesis of learning objectives. Each stage ends with a mandatory stop: the teacher explicitly approves before the next stage begins. Each stage includes validation checkpoints to ensure accuracy before advancing.

### The Output
Validated learning objectives that reflect **actual teaching** (not just planned curriculum), providing the foundation for systematic assessment design in M2.

## THE SIX-STAGE PROCESS (STAGE 0–5)

M1 consists of six sequential stages — Stage 0 (material analysis) followed by five dialogue stages (1–5) — each documented in detail in separate files:

### Stage 0: Material Analysis (m1_0_stage0_material_analysis)
**Duration:** 60-90 minutes
**Nature:** Claude's independent analysis before dialogue
**Purpose:** Understand instructional content to prepare for dialogue
**Output:** Initial content architecture with priority tiers, example catalogue, misconception notes
**Required File:** m1_0_stage0_material_analysis

### Stage 1: Initial Validation (m1_1_stage1_validation)
**Duration:** 20-30 minutes
**Nature:** Validation dialogue
**Purpose:** Confirm Stage 0 analysis accuracy before detailed dialogue
**Output:** Validated priority structure with teacher corrections
**Required File:** m1_1_stage1_validation

### Stage 2: Emphasis Refinement (m1_2_stage2_emphasis)
**Duration:** 30-45 minutes
**Nature:** Dynamic dialogue with adaptive inquiry
**Purpose:** Refine understanding of content priorities and pedagogical rationale
**Output:** Refined understanding of emphasis patterns, teaching approaches, assessment priorities
**Required File:** m1_2_stage2_emphasis

### Stage 3: Example Catalogue (m1_3_stage3_examples)
**Duration:** 20-30 minutes
**Nature:** Systematic inquiry about effective examples
**Purpose:** Document instructional examples that worked well
**Output:** Organised catalogue of examples by topic with effectiveness notes
**Required File:** m1_3_stage3_examples

### Stage 4: Misconception Registry (m1_4_stage4_misconceptions)
**Duration:** 20-30 minutes
**Nature:** Systematic inquiry about student errors
**Purpose:** Document common misconceptions for distractor development
**Output:** Registry of misconceptions with severity ratings and correction strategies
**Required File:** m1_4_stage4_misconceptions

### Stage 5: Scope Boundaries & Learning Objectives (m1_5_stage5_objectives)
**Duration:** 45-60 minutes
**Nature:** Clarification dialogue + synthesis
**Purpose:** Finalise scope and synthesise validated learning objectives
**Output:** Complete learning objectives with evidence, scope boundaries, transition notes for M2
**Required File:** m1_5_stage5_objectives
